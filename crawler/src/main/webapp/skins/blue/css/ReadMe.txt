文件简介

一、公共样式文件

公司所有整体项目公共样式文件：common.css

项目公共样式文件：comm.css

项目中弹出框及弹出框内部内容样式：dialog.css

引用easyui框架所自带的css文件：easyui.css

二、独立页面样式文件

项目头部及尾部样式文件：header.css

项目登录页面样式文件：login.css

项目登录成功后为系统管理页面，其样式文件夹为：system_css
	
    （1.文件夹中subFrame.css文件为页面主框架内容的样式文件
      2.文件夹中subFramebmgl.css等类似带有导航名称简写的css文件为主框架中各页面的样式文件
      3.文件夹中subFramextszSubFrame.css文件为主框架中再次使用框架后其中内容的样式文件）

三、图片

html文件中直接使用的图片直接放在images中

easyui中使用到的图片放在easyui中
